# MinerU 오프라인 배포 가이드

본 문서는 MinerU를 사내 오프라인 환경에서 배포하고 운영하기 위한 가이드를 제공합니다.

## 주요 특징
- **오프라인 모드**: 모든 모델을 로컬 디렉토리에서 로드하며, 외부 인터넷 연결 없이 동작합니다.
- **CPU 최적화**: VLM을 제외한 모든 모델은 CPU에서 ONNX 런타임을 사용하여 최적화된 추론을 수행합니다.
- **사용량 추적**: SQLite (WAL 모드)를 사용하여 API 호출 내역을 기록합니다.
- **한국어 최적화**: 기본 OCR 설정을 한국어로 고정하여 한글 문서 분석 성능을 극대화했습니다.

## 디렉토리 구조
- `models/`: 오프라인에서 사용할 모델 파일 보관
- `docs/ko/`: 한국어 기술 문서
- `scripts/`: 모델 다운로드 및 유틸리티 스크립트
- `usage.db`: 사용량 기록 데이터베이스 (SQLite)

## 빠른 시작
1. 모델 다운로드 (인터넷 가능 환경에서 선행)
   ```bash
   python3 scripts/download_models.py --dir ./models
   ```
2. 서비스 기동
   ```bash
   ./start.sh prod
   ```
3. 서비스 중지
   ```bash
   ./stop.sh
   ```
